## Get my background

내가 갖고있는 모든 배경을 조회합니다.

## Request
### HTTP METHOD : `GET`

### url : `https://render.gitanimals.org/users/{username}/backgrounds`

### PathVariable
- {username}: 사용자의 이름

## Response

```json
{
  "id": "1",
  "name": "devxb",
  "backgrounds": [
    {
      "type": "SNOWY"
    },
    {
      "type": "WHITE"
    }
    ...
  ]
}
```
