## Get background

모든 구매가능한 배경을 조회합니다.

## Request

### HTTP METHOD : `GET`

### url : `https://api.gitanimals.org/shops/backgrounds`

### Response Body

200 OK

```json
{
  "backgrounds": [
    {
      "type": "SHOP",
      "price": "100000"
    },
    {
      "type": "SNOWY",
      "price": "100000"
    },
    ...
  ]
}
```
