# Get all guild icons

존재하는 길드 아이콘들의 이름을 반환합니다.


## Request
### HTTP METHOD : `GET`
### url : `https://render.gitanimals.org/guilds/icons`

## Response

200 OK

```json
{
  "icons": [
    "https://static.gitanimals.org/guilds/icons/GUILD-1",
    "https://static.gitanimals.org/guilds/icons/GUILD-2",
    ...
  ]
}
```
