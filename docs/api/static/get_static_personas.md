# Get static personas

persona들의 정적 파일을 조회합니다.

## Request
### HTTP METHOD : `GET`
### url : `https://static.gitanimals.org/personas/{persona-type}`

### Path variable
- persona-type: 정적파일을 조회할 persona의 tytpe을 지정합니다.
